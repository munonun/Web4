# Web4

Experimental MVP implementing Web4 local-record + ACK-based settlement.

## Status

- v0.0.2 (Alpha)
- Not production-ready
- No network, copy-paste protocol

## Core ideas

- No global ledger
- Local contracts (IOU) as state
- Only the creditor can ACK & close
- SHA3-based signing
- No consensus, no ordering

## Commands

- keygen
- open
- list
- close
- ack

## Warning

This is a research prototype. Do NOT use with real value.
